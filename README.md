ProjectName

IPrescribe

Description

iPrescribe is an admin-focused assessment dashboard designed to help administrators manage and review user data effectively.

Features

Login: Secure admin login to access the dashboard.

Dark Mode: Toggle between light and dark themes for better usability.

Remember Me: Option to stay logged in for convenience.

Search: Search for users by name, location, or email.

Sortable Tables: Easily sort patient data by column headers.

Installation

Clone the repository:

git clone https://github.com/Stunner05/IPrescribe-Project.git

Navigate to the project directory:

cd your-repo

Install dependencies:

npm install

Start the project:

npm start

Usage

Log in as an admin to access the dashboard.

Use the search functionality to find users by name, location, or email.

Toggle between light and dark mode as needed.

Utilize the “Remember Me” option for convenience.

Sort tables to organize data effectively.

Bonus Features

Dark Mode: Switch between light and dark themes for improved readability.

Remember Me: Keep the admin logged in for future sessions.

Search Functionality: Quickly find users by name, location, or email.

Sortable Tables: Organize data by sorting columns for better management.
